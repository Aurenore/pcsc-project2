Virtual function, overriden in the daughter classes, setting the coefficients values b[i][j] of the equations to solve .

Set the matrix B of coefficients which define the equations to solve for each order. B is composed of 5 rows and 6 columns, the first lign corresponding to the coefficients included in the equation for order 1 and the last lign for order 5.
