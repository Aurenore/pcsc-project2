void AdamsMoultonSolver::SetB ( )
overrideprotectedvirtual

Virtual function, overriden in the daughter classes, setting the coefficients values b[i][j] of the equations to solve .

Set the matrix B of coefficients which define the equations to solve for each order. B is composed of 5 rows and 6 columns, the first lign corresponding the coefficients included in the equation for order 0 and the last lign for order 4.

Virtual function, overriden in the daughter classes, computing the numerical solution of the ODE.

Adams Moulton methods for the scalar ODE in the form y'(t)=f(y,t). The Newton method is used to solve the nonlinear equation at each time t.
